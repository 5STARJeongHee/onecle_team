-Smash.gg는 전 세계에서 게임을 즐기는 플레이어들과 대회 주최자들이 이용하는 이스포츠 플랫폼이다.

-Smash는 Firebase 실시간 데이터베이스에 저장된 데이터의 변경사항에 대응하도록 Cloud 함수를 재빠르고 간편하게 설정했다.

-Smash는 Cloud 함수를 통해 자동으로 사용자에게 대전 가능한 경기를 알리거나 플레이어 간의 분쟁이 생겼을 때 대회 운영자에게 통보하게 된다.

-이때 Cloud 함수를 사용하여 브라우저 푸시 알림을 전송하기 위해 사용된 것이 Firebase 클라우드 메시징 (FCM)이다.
